## ✨ Why OmniRust?

- **Blazing Fast**: Written entirely in Rust — zero runtime overhead, memory safe, starts in < 10ms
- **All-in-One**: Dozens of essential developer tools packed into **a single CLI binary**
- **CLI First**: Clean subcommand design with shell completions, colorful output, and parallel processing
- **Cross-Platform**: Native support for Linux, macOS, and Windows (including musl static builds)
- **Extensible**: Plugin system for easy community contributions
